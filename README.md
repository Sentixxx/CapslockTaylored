# CapslockMagic

Capslock Magic 是一个基于 Autohotkey 开发的改良式快捷键映射的方案。将 ⇪ CapsLock（大写锁定键）改造为一个强力的功能修饰键（✱ Hyper ），奇迹般地提高操作效率与生产力。

它积极响应变化，源码随时可改，引导你进化出适合自己的快捷键风格。
